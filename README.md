Clock Application

Description:
Displays a clock ui that displays time in 24 and 12 hour formats

The program does the requirements as outlined in the 210 Programming languages course. This is implementing string tool usage.

Improvements would be to make it tick based on UTC time. 

The most difficult part was thinking through a 24 hour or 12 hour algorithm. I decided that converting to seconds and then converting back would be the easiest thing to do. This presented itself as challenges. To improve this, using time values would be more useful than having to build them from scratch.

Functionalizing parts of an application was useful and learning to think through algoritms to solve the problem helped me out and will carry ways to solve similar situations in the future.
